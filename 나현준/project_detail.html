프로젝트 1: 반응형 온라인 쇼핑몰 구축

최신 웹 표준을 준수하고 사용자 경험(UX)을 극대화한 반응형 온라인 쇼핑몰 웹사이트입니다. 다양한 디바이스 환경에 최적화된 레이아웃을 제공하며, 상품 관리, 주문 처리, 결제 시스템 연동 기능을 구현했습니다.

  • 사용 기술: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+), React.js (혹은 Vue.js/Angular), Node.js (Express), MongoDB (혹은 MySQL), RESTful API, Webpack
  • 기여도: 100% (프론트엔드/백엔드 개발, 데이터베이스 설계, API 연동)
  • 주요 기능: 상품 목록/상세 페이지, 장바구니, 주문/결제 프로세스 (UI), 사용자 인증, 검색 필터링, 관리자 페이지 (UI)
