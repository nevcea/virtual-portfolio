Q&A - 연락처 페이지 개발 경험

Q1. 어떤 역할을 맡았고, 무슨 작업을 했나요?
저는 포트폴리오 웹사이트에서 '연락처(Contact)' 페이지 제작을 맡았습니다. 사용자가 이메일을 보낼 수 있는 폼을 구현하고, 시각적으로도 깔끔하게 보이도록 HTML과 CSS를 활용해 디자인했습니다.
Q2. 작업하면서 어떤 어려움이 있었나요?
폼 데이터를 실제로 전송하는 기능에서 어려움이 있었습니다. 백엔드 없이 메일을 보내기 위한 방법을 찾는 과정에서 시행착오가 많았습니다.
Q3. 그 어려움을 어떻게 해결했나요?
EmailJS 같은 외부 서비스를 활용해 백엔드 없이도 메일을 전송할 수 있다는 것을 알게 되었고, 관련 문서를 보며 구현했습니다. 디버깅을 여러 번 하면서 문제 해결 능력이 향상되었습니다.
Q4. 이 작업을 통해 어떤 점에서 성장했다고 느끼나요?
단순히 보이는 페이지를 넘어서, 사용자 경험과 실질적인 기능 구현까지 고민하게 되면서 문제 해결 능력과 개발 실력이 성장했다고 느꼈습니다.
Q5. 이번 경험을 앞으로 어떻게 활용하고 싶나요?
이번 경험을 바탕으로 더 복잡한 기능에도 도전하고 싶고, 사용자 친화적인 웹 페이지를 만드는 개발자가 되기 위해 계속해서 배우고 싶습니다.
